## Assignment rule -> Implementation mapping (1:1)

1. **Simulación de 4 robots AGV en mapa 2D (nodos QR)**
   - `SimulationConfig(n_robots=4)` in `src/agv_sim/models.py`.
   - 2D grid graph in `create_grid_graph(width, height)` (`src/agv_sim/graph_utils.py`).

2. **Inicio en bases y búsqueda de carrito asignado aleatoriamente**
   - Bases at map corners in `_init_entities()` (`src/agv_sim/simulation.py`).
   - Random assignment in `_allocate_idle_robots()` with seeded RNG.

3. **Carrito cambia al color del robot asociado (robots con color único)**
   - Color coupling on assignment (`cart.base_color = r.color`).
   - Reset to original color (`gray`) after delivery.

4. **Rutas con algoritmos de grafos**
   - Operational routing modes: `random_shortest`, `dijkstra`, `bfs`.
   - Implemented in `_shortest_path()` and `graph_utils.py`.

5. **Movimiento aleatorio hacia nodo final**
   - `random_shortest_path(...)` randomizes tie-breaks among equivalent shortest paths.
   - Keeps intelligent shortest-route behavior while introducing randomness.

6. **Al alcanzar carrito: llevar a destino, liberar, volver a base, cargar**
   - State machine:
     - `TO_CART -> TO_DESTINATION -> RETURN_TO_BASE -> CHARGING -> IDLE_AT_BASE`
   - Transition events logged for charging-cycle evidence.

7. **Rotaciones en pasos de 10°**
   - `rotation_step_deg=10.0` in config.
   - Applied in `_rotate_then_move()` before node movement.

8. **Temas de grafos exigidos (profundidad, ruta más corta, todas y al menos una ruta)**
   - DFS evidence: `dfs_any_path(...)`.
   - Shortest route evidence: `dijkstra_path(...)`.
   - At-least-one-route evidence: non-empty path checks.
   - Multiple-route evidence: `all_simple_paths_limited(...)` sample.
